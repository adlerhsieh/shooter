操作方式
=========
1. 先在urllist.csv 中，準備要抓取的網站截圖編號和網址，格式可參考 urllist.example.csv
2. 在命令列中執行 php app.php，程式即會開始抓圖，抓取完的圖會放在 images的目錄底下
3. 中間如果發生問題，可以重新執行，或者將 urllist.csv 中已經抓過的網址刪去後再執行，可以加快執行過程
4. 流程可參考影片: https://youtu.be/0iAXV_3ch6M 

注意事項
===========
1. 程式需要具在命令列執行PHP的環境
2. 程式在PHP 5.6.x的版本中開發完成，需要以這個版本之後的才能確保順利執行
3. 網站截圖抓取程式依賴　http://www.page2images.com/　這個網站所提供的 API，在未付費的情況下，每日抓取的數量有上限，另外未付費也不能抓取 https 開頭的網站
4. 如果要使用付費版本，可至該網站申請帳號並付款，取得新的 API id 後，可以修改 core 目錄底下的 config.php 裡面的 id，才能換成付費的帳號
